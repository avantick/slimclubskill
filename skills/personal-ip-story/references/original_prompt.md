# 个人IP故事挖掘

以下内容整理自用户提供的《AI提示词.pdf》对应章节，用作执行该 Skill 时的原始提示词参考。

```text
1. 判断我是否具备真实表达动机(而不是跟风做内容) 
2. 提炼我的 “ 核心人物特质 ” (一句话定义我是谁) 
3. 从我的经历中挖掘可讲的故事素材(按结构分类) 
4. 筛选出最有 “ 爆点潜力 ” 的故事 
5. 帮我建立一个可长期复用的 “ 个人故事结构 ” 
 
## 输入信息 
 
我会提供: 
 
- 我的背景经历(越真实越好) 
- 我现在在做的事情  / 行业 
- 我想做内容的目的(涨粉  / 变现  / 建立信任等) 
- 我想到的一些零散故事(如果有) 
- 我的目标受众是谁 
 
## 约束条件

1. 不允许直接帮我 “ 写故事 ” ,必须先挖掘和判断

2. 所有故事必须围绕 “ 核心人物特质 ” 展开 
3. 必须筛掉: 
   - 没有情绪 
   - 没有转折 
   - 没有价值的经历

   - 给力量

## 输出结构 
1. 核心人物特质提炼(最关键) 
- 用一句话定义我是谁 
- 我的核心特质是什么    
  如:极度主动  / 极度理性  / 极度反叛等 
- 为什么是这个特质,结合经历说明 
 
2. 故事素材挖掘(结构化) 
 
请按三类整理: 
1  源起叙事:我为什么走上这条路。 
2  转折叙事:改变我人生的关键节点。 
3  洞察叙事:我对行业  / 人性的独特判断。 
 
3. 爆点故事筛选 
 
从所有素材中筛选: 
- 最有情绪张力的 3 个故事 
- 最有共鸣的 3 个故事 
- 最能建立专业信任的 3 个故事 
并说明为什么值得讲。 
 
4. 个人 IP 叙事结构(长期可复用) 
 
- 我的故事主线是什么 
- 不同内容可以如何围绕这条主线展开 
- 如何做到 “ 讲 100 个故事,但都在讲同一个人 ”
```
