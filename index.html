仓库中的 Skill

两个 Skill 形成一条清晰链路：先中立跟踪热点，再结合产品库生成选题卡。

上游 · 热点数据 中立抓取

瘦吧热点跟踪skill

skills/shouba-hotspot-tracking

从 UApiPro 获取小红书热榜，输出原始、可追溯的热点列表。它只负责热点跟踪，不做产品匹配、用户痛点推断、机会判断或内容建议。

  • 获取 UApiPro 小红书热榜
  • 保留排名、标题、热度值、链接、抓取时间
  • 输出中立热点跟踪报告
  • 作为后续产品匹配和选题卡生成的输入

瘦吧小红书选题卡skill

skills/shouba-xhs-topic-card

  • 接收热点数据和产品库
  • 评估话题、场景、痛点和产品植入自然度
  • 输出小红书选题卡
  • 为内容智能体提供结构化输入

推荐使用流程

这个流程可以用于培训演示，也可以作为内容团队的日常小红书选题工作流。

1

跟踪热点

使用“瘦吧热点跟踪skill”从 UApiPro 获取当前小红书热榜，只保留原始热点数据。

2

输入产品库

提供产品名称、卖点、人群、场景、用户痛点、可表达利益点和风险边界。

3

生成选题卡

使用“瘦吧小红书选题卡skill”评估热点与产品匹配度，并生成标准选题卡。

4

交给智能体

把选题卡交给小红书内容智能体，继续生成标题、封面、正文和评论引导。

下载后如何安装

下载 zip 后，可解压到 WorkBuddy 用户 Skill 目录，或按团队内部流程导入。

方式一：通过页面下载

点击上方“下载安装包”，下载对应 zip 文件，再按照 WorkBuddy 的 Skill 导入流程安装。

下载热点跟踪 Skill 下载选题卡 Skill

方式二：手动复制源码目录

cp -R skills/shouba-hotspot-tracking ~/.workbuddy/skills/
cp -R skills/shouba-xhs-topic-card ~/.workbuddy/skills/

复制后如 WorkBuddy 未立即识别，可重启 WorkBuddy 或重新打开对话。
